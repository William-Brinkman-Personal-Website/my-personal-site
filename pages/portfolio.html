Projects

Calculator App

This simple calculator app allows users to perform basic arithmetic operations (addition, subtraction, multiplication, and division) on two numbers. Users can enter the numbers using an on-screen keypad, and the app displays the result of the calculation in a separate field. The app uses JavaScript to handle the calculations and update the display in real-time. It includes a clear button to reset the calculation and start over.

Weather App

This weather app allows users to enter a city or zip code and retrieve the current weather conditions for that location. It uses an API to fetch weather data, and displays the temperature, humidity, wind speed, and other relevant information in an easy-to-read format. The app includes a search bar for entering the location, and updates the display automatically as the user types. It also includes a feature for displaying the weather forecast for the next few days. The app is designed using HTML, CSS, and JavaScript, and is responsive to different screen sizes.
